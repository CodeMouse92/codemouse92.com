Projects█

I've worked on a lot of diverse projects, both independently and through my software company. Here's a few highlights...

  • dicebox: Roll dice, flip coins, and simulate other decision-making methods.
  • dkim_manage: A script to automate most tasks associated with OpenDKIM key rotation.
  • magichosts: Automatically detects a target network and turns on/off LAN routing in your /etc/hosts file accordingly. (I use this as a part of my daily system configuration.)
  • Omission: A game where you have to find the missing letter in a passage. (Work in progress...currently refactoring and switching to PySide2.)

  • PawLIB: Trilean: An atomic C++ data type with three states: true, false, or maybe.
